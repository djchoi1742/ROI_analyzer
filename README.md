# ROI_analyzer
- Program for image texture analysis
- Main features
  - View basic statistics
    - Mean, Standard deviation
    - Median, Mode
    - Minimum, Maximum
    - Skewness, Kurtosis
    - Entropy, Percentile
  - View histogram of textures
    - View basic statistics after adjust texture range
